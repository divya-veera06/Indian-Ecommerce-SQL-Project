# Indian Ecommerce SQL Project

This is an end-to-end SQL project for an Indian ecommerce store.  
It includes database schema, sample data, and queries from basic to intermediate level.

## Files included
- schema.sql → create tables
- data.sql → insert sample data
- queries.sql → all SQL queries

## How to use
1. Open `schema.sql` in MySQL Workbench → run to create database and tables
2. Open `data.sql` → run to insert sample data
3. Open `queries.sql` → run queries to analyze the data
